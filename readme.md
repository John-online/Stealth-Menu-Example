# Stealth Menu Example

This project is a simple example menu built for Project Stealth's Isolated executor API.

It uses a Lua script to create and control a DUI menu, while a small local Node.js server hosts the HTML interface.

The example demonstrates basic menu toggling and arrow-key navigation messaging between Lua and the DUI page.
